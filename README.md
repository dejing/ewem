## 平台介绍
#### 目前二维码应用非常广泛，生产制造、品牌溯源、农产品、移动巡检、快消品。大企业根据自己的经营特点定制开发。但中小企业很难找到符合自身需求的标准化产品，定制化开发的高成本也让中小企业望而却步。
#### 故开发一套应用一物一码的轮子，欢迎各位 Watch、Star。

## 技术选型
* 前端采用Vue、Element UI。
* 后端采用Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
